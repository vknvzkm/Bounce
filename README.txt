Игра-проект: Bounce

Суть проекта: Bounce(Bouncing Ball) — это классическая аркадная игра, которая имеет простой и увлекательный геймплей.
Игрок управляет шариком, который прыгает по платформам и собирает бонусы на своем пути. Помимо этого, игрок будет сталкиваться с различными препятствиями, такими как пропасти, острые шипы и падающие ящики, которые могут убить его шарик, поэтому следует аккуратно проходить через все эти препятствия.

Цель игры: Пройти через различные уровни, избегая препятствий и достигая выхода.

Правила игры
	⁃	Управляйте мячом, используя клавиши управления (влево, вправо, вверх для прыжка).
	⁃	Избегайте столкновений с шипами и другими препятствиями.
	⁃	Достигните выхода, чтобы завершить уровень.

Запуск игры:
Для запуска игры необходимы
-Python
-модуль pyGame
-pyCharm 

Для оформления использован шрифт ARCADECLASSIC, для визуального оформления используются изображения, которые расположены в папке assets и tiles. В папке levels находится уровень игры. При необходимости можно создавать и добавлять новые уровни.

Для того, чтобы открыть игру, необходимо в pyCharm открыть main.py

Используемые библиотеки и функции:
Для создания игры использован язык программирования Python с использованием библиотеки Pygame.
Некоторые из ключевых функций и классов, используемых в коде главной функции main.ru:
- pygame.init( ): инициализация Pygame.
- pygame.display.set_mode ( (WIDTH,HEIGHT) ): создание окна игры.
- pygame.time.Clock(): создание объекта Clock для управления FPS.
⁃ Загрузка изображений для игровых объектов и интерфейса.
⁃ Создание объектов классов Ball,World, Message n Button для управления игровыми элементами.
⁃ Работа с группами спрайтов для управления объектами на уровне.
⁃ Обработка событий клавиатуры для управления персонажем.
⁃ Логика переключения между экранами игры (главное меню, игровой экран, экран рестарта).
